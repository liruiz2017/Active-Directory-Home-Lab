# Lessons Learned

Throughout this Active Directory Home Lab, I gained practical experience deploying and administering a Windows enterprise environment.

Key lessons learned include:

- DNS is critical for Active Directory functionality.
- Organizational Units simplify administration and policy management.
- Security groups make permission management scalable.
- Group Policy provides centralized management of Windows settings.
- NTFS permissions and Share permissions work together to secure resources.
- PowerShell greatly improves administrative efficiency by automating repetitive tasks.
- Proper documentation and screenshots are valuable for troubleshooting and portfolio development.

This project strengthened my understanding of enterprise Windows administration and provided hands-on experience that closely mirrors real-world IT environments.
