# Day 1 Journal

## Objective

Deploy a functional Active Directory environment using Windows Server 2022 and VMware Fusion.

---

## Completed Tasks

- Installed Windows Server 2022
- Configured DC1
- Installed Active Directory Domain Services
- Created the ruizlab.local domain
- Configured DNS
- Created Organizational Units
- Created department users
- Created security groups
- Installed Windows 11 Enterprise
- Joined PC01 to the domain
- Verified domain authentication
- Created VMware snapshots
- Began GitHub documentation

---

## Challenges

- VMware Easy Install issues
- EFI boot problems
- DNS configuration preventing domain join

Each issue was successfully resolved through troubleshooting.

---

## Skills Practiced

- Windows Server Administration
- Active Directory
- DNS
- Domain Join
- User Administration
- Organizational Units
- Security Groups
- VMware
- Documentation

---

## Reflection

Completing both the Domain Controller and the Windows 11 client in a single day provided valuable hands-on experience with enterprise identity management. The troubleshooting process reinforced the importance of networking, DNS, and careful system configuration.
