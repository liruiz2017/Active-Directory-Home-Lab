# Troubleshooting

## Common Issues Encountered

| Issue | Resolution |
|------|-------------|
| VMware Easy Install failed | Installed Windows manually |
| EFI Boot Error | Corrected virtual hardware configuration |
| Client could not join the domain | Configured DNS to point to the Domain Controller |
| Group Policy not applying | Ran gpupdate /force |
| Access Denied on shared folders | Corrected NTFS and Share Permissions |
| PowerShell command error | Corrected cmdlet syntax and parameters |

---

## Lessons Learned

Troubleshooting is one of the most important skills in systems administration. Throughout this project, multiple issues involving networking, Active Directory, permissions, and PowerShell were resolved through testing and verification.

Each problem improved understanding of enterprise Windows administration.
