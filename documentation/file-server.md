# File Server

## Overview

This lab demonstrates the creation and management of Windows file shares using Active Directory security groups and NTFS permissions.

---

## Shares Created

| Share | Purpose |
|--------|----------|
| Finance | Finance department documents |
| HR | Human Resources documents |
| IT | IT department resources |
| Sales | Sales department files |

---

## Permissions

Each shared folder was configured using both:

- Share Permissions
- NTFS Permissions

Access was assigned according to department membership.

---

## Testing

The following tests were performed:

- Authorized users successfully accessed their department shares.
- Unauthorized users received an Access Denied message.
- Group membership controlled access to resources.

---

- Share Permissions
